# RECELL-AI

Battery Classification System for KIWIE Korea.

## Components
- **Firmware (STM32F411):** Sensor reading and actuator control.
- **AI (Jetson Orin Nano):** Battery judging and classification.

## Structure
- `/firmware`: STM32 source code.
- `/jetson`: Jetson AI models and inference code.
- `/research`: Documentation and research materials.
- `/scripts`: Setup and utility scripts.
